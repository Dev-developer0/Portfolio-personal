# Portfolio Website

A modern, responsive portfolio website for Dev Sharma, a web developer specializing in frontend development, backend systems, UI/UX design, and full-stack development.

## Features

- **Responsive Design**: Optimized for all device sizes using modern CSS techniques
- **Smooth Animations**: CSS transitions and JavaScript-powered scroll animations
- **Interactive Elements**: Hover effects, sticky project cards, and fade-in animations
- **Clean Architecture**: Separated HTML, CSS, and JavaScript files for maintainability
- **Modern UI**: Dark theme with gradient accents and professional typography

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript (ES6+)**: Interactive functionality and scroll-based animations
- **Google Fonts**: Kanit font family for typography
